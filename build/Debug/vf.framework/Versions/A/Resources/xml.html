Experiment Markup Language

Experiment data and applications settings are persisted in an XML format, Reify Experiment Markup Language (file extension .rfyeml). See official DTD. One experiment file can contain both settings and analysis data. Settings are encapsulated in one or more <setting-category> elements. Analysis data is encapsulated in a single <experiment-data> element. 
<experiment>
    <setting-category>
        <!-- Application settings -->
    </setting-category>
    <experiment-data>
        <!-- Analysis data -->
    </experiment-data>
</experiment>
