Visible Release Notes

  

Visible Myosight 2.0

January 16th 2006

System Requirements

  • Disable computer and hard drive sleep from System Preferences. Video capture process will fail if the system goes to sleep. Display sleep does not have to be disabled.
  • Do NOT run any other user applications during video capture.
  • Video analysis requires large amounts of memory at least 1 GB of RAM is recommended for best performance.

Licensing

  • Visible license is a single computer license.

Limitations

  • Visible 2.0 Myosight is a beta version.

Known Issues

  • Analysis selection "Paced Myocyte" needs to be selected even though it is the only option.
