107  /* genEightBitImg - Convert kinetic image, composed of floats, into
108  * an unsigned eight bit image. In the 1st case, the destination
109  * pixels will be centered around the kinetic image's mean, with
110  * sdClamp specifying the number of standard deviations to map out
111  * to the min (pixel value 0) and max (pixel value 255) values. If
112  * sdClamp is < 0.0, then the kinetic image's entire dynamic range
113  * is used. Note that, in this case, the kinetic image must contain
114  * at least 2 pixels.
115  *
116  * In the 2nd case, the caller specifies an arbitrary mapping. The
117  * values in map should be in strictly ascending order. For each
118  * kinetic pixel value P, (map[N] <= P < map[N+1]) ==> P maps to N+1
119  * (as you would expect, (P < map[0]) ==> P maps to 0 and (P >=
120  * map[LAST]) ==> P maps to LAST + 1). Up to the 1st 255 elements of
121  * the map are used.
122  */
