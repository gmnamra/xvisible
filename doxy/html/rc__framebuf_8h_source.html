368  /* For cached frames, the cache controller is called to mark this
369  * frame as available for reuse. The frame's data is retained in
370  * cache, so that a subsequent call to lock() can retrieve the
371  * data. Caching is controlled using a least-recently-used
372  * algorithm.
373  *
374  * WARNING: Any rcFrame related memory references, such as the
375  * pointer returned by rawData, or a reference to the rcFrame
376  * object itself is only valid while the frame is locked. To use
377  * these values again, it is necessary to relock the frame and
378  * then reread these values.
379  *
380  * Note: This is a NOOP for uncached frames.
381  */
