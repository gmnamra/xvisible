1 /******************************************************************************
2  * @file rc_setting.h
3  *
4  * This file declares classes that support an abstract representation
5  * of settings used to control the operation of various parts of the
6  * application.
7  *
8  * An individual setting specifies a meta-data tag name,
9  * a display name, and description, as well as a generic specification
10  * of the type of UI widget that should display and control it.
11  *
12  * A number of settings can be grouped into a settings category, which
13  * specifies a meta-data tag name, display name and description.
14  *
15  * A pure-virtual interface class is specified that is implemented by
16  * classes that have settings they wish to have controlled from the UI.
17  *
18  ******************************************************************************/
