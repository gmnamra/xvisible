4 * This file contains platform independent declarations for supporting
5 * threads.

7 * A thread can be set up in one of two ways.

9 * An object can implement rcRunnable, instantiate an rcThread with

109 // Provides support for thread based mutexes. Typically, users should

113 class RFY_API rcMutex
114 {
115 public:
116  // Create a mutex, Setting invertProtect to true eliminates
117  // priority inversion problems. Setting errorCheck to true
118  // causes an error to occur whenever either a thread attempts
119  // to unlock a mutex it has not already locked or a thread
120  // attempts to lock a thread that it has already locked. tp
121  // determines what priority locking thread will run at. It is
122  // only used when invertProtect is true.
123  //
