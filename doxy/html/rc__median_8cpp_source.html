13 /*
14  A reasonably efficient C implementation of the true 3x3 median
15  filter. This routine uses the TI algorithm as documented in the
16  DSP example code.
17 
18  1. Compute the min, max and median for each column
19 
20  2. Find the minimum of the 3 column max's, the maximum of the
21  3 column min's, and the median of the 3 column medians.
22 
23  3. The median of the 3x3 is then found as:
24 
25  med(minMax, maxMin, medMed)
26 
27  This is a clever algorithm because no 9 point sort is required
28  and only 3 pt medians are needed. Additionally each column term
29  can be reused over three neighborhoods.
30  */
