Detailed Description

C++ wrapper around the glob() system call.

(extracted from my rpm_find_latest project, 2004/04)

It works similar to an STL container: Pass it a list<> of paths and use the iterators to touch each item, as expanded by the system call.

This class is named "rcGlob" because the name "Glob" conflicted with an existing class somewhere (but not within this project).
