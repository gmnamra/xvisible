37  /* Checks to see if shared memory can be acquired. If it cannot,
38  * and wait is TRUE, then sleep until shared memory is available.
39  *
40  * In either case, if shared memory is available, it is locked from
41  * use by the corresponding rcSharedMemoryUser.
42  *
43  * Returns a pointer to shared memory if it is available. Otherwise,
44  * NULL is returned. Check err to see reason for NULL return value.
45  *
